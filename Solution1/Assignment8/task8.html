Search Bacterial Genomes

Search for complete bacterial genomes from NCBI matching your regular expression. Multiple patterns can be entered separated by spaces.

Organism Name (Regex)
Supports full Python regex syntax. Examples:
^Escherichia.*coli$   Mycobacterium   Bacillus subtilis
Quick examples
Escherichia.*coli Mycobacterium tuberculosis Bacillus subtilis

About This Tool

Queries the NCBI genome reports for complete bacterial genomes. Returns matching organism names, genome sizes, and GC content. Only complete (non-draft) assemblies are included.
