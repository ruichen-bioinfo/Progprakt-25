Task 5 — Gruppe 11 · LMU Programmierpraktikum WS2025/26

Retrieve FASTA by Accession Number

Enter a UniProtKB/SwissProt accession number to retrieve the full FASTA sequence entry.

Accession Number
Format: one or two letters followed by five digits (e.g. P12345, Q9Y6K9)
Quick examples
P12345 Q9Y6K9 P00533 P68871

About This Tool

Queries the UniProtKB REST API and returns the sequence in standard FASTA format. Results are displayed directly in the browser.
