Pipeline Overview

Upload a genome FASTA file and its feature table to extract all annotated protein sequences via a three-step pipeline.

genome2orf.py → dna2mrna.py → mrna2aa.py

1 Genome FASTA File

Upload nucleotide genome sequence
Nucleotide sequence in FASTA format. Accepted extensions: .fasta .fa .fna

2 Feature Table

Upload CDS feature annotations (tab-separated)
Feature table from NCBI containing CDS annotations. Tab-separated format with columns for start, stop, strand, and locus tag.
