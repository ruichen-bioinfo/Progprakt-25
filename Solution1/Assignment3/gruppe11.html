Git Kurzanleitung

Hier finden Sie eine kurze Einführung in Git:

  • git clone <repository-url> – Repository lokal kopieren
  • git status – Status der Arbeitskopie anzeigen
  • git add <datei> – Datei zum Commit vormerken
  • git commit -m "Nachricht" – Änderungen lokal speichern
  • git pull – Änderungen vom Server holen
  • git push – Eigene Änderungen hochladen
