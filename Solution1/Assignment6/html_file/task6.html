SwissProt Keyword Search

Task 6 — Gruppe 11 · LMU Programmierpraktikum WS2025/26

Search by Keywords

Enter one or more keywords to search the SwissProt database. Multiple keywords are combined with AND logic.

Keywords (space-separated)
All keywords must be present in the entry. Example: human insulin receptor
Quick examples
membrane kinase insulin receptor human ribosome Escherichia

About This Tool

Searches UniProtKB/SwissProt using the full-text search API. Returns a list of matching entries with their accession numbers, names, and organism information.
